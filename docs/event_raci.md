# EPA Event RACI Matrix

| Deliverable / Decision | EA | EPA | CRA | Security | Facilities |
| --- | --- | --- | --- | --- | --- |
| Define event objectives & success metrics | C | R | C | I | I |
| Draft agenda & segment owners | C | R | I | I | I |
| Security + privacy content review | I | C | C | R | I |
| Venue booking & onsite logistics | I | R | I | I | A |
| AV/catering coordination | I | R | I | I | A |
| Promotion copy (emails, intranet, signage) | C | R | A | I | I |
| Pilot signup + registration workflow | R | C | I | I | I |
| Risk log + contingency planning | R | R | C | C | C |
| Day-of runbook + staffing | A | R | C | I | C |
| Post-event survey + enablement follow-up | R | C | C | I | I |

Legend: **R** = Responsible, **A** = Accountable, **C** = Consulted, **I** = Informed.
